# Ecommerce-Sales-Analysis
I just wrapped up an E-commerce Sales Analysis project! 🚀
​I built an interactive dashboard to track 37K in profit and analyze customer behavior across India. Key takeaway: Maharashtra is our strongest market, and COD remains the king of payment modes at 44%.
​Check out the repository for the full breakdown of how I turned raw transactional data into these visual insights.
​Quick Tip for your GitHub: If you can, export the data to a .csv or .xlsx and include a small "sample_data" folder in your repository so people can see the structure you worked with!
